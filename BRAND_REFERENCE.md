# RootWork Framework Brand Reference

## Brand Essence
- **Positioning:** Dual-purpose pedagogy weaving academic rigor with healing-centered, biophilic practice
- **Personality:** Grounded • Compassionate • Practical • Scholarly • Community-centered
- **Tone:** Plain language, invitational ("we"/"our"), non-punitive, action-oriented
- **Narrative Pillars:** Relationships, Routines, Relevance, Rigor, Reflection (5Rs)

## Color System

| Color | Hex | Usage |
|-------|-----|-------|
| **Evergreen** | #082A19 | Primary background |
| **Deep Canopy** | #001C10 | Near-black for deep backgrounds |

| Color | Hex | Usage |

- Green family: 60%
- Neutrals: 30%

- Use official vector files only (SVG/AI)
- **Preferred:** Gold Leaf #D4C862 on Evergreen #082A19

- **Light bg:** Evergreen #082A19 on Canvas Light #F2F4CA
- Minimum width: 180px (web), 0.9in (print)
- Clear space: Equal to sprout emblem diameter

## Layout
- **Spacing system:** 8px increments
- **Grid:** 12-column (posters), 8-column (web/slides)
- **Max line length:** 65-85 characters

## LMS Design Philosophy (Three-Platform Architecture)

### Core Platforms
1. **RootWork Tablet** - Structured, sequential learning
2. **RootWork LMS** - Dashboard for teachers/parents/admins
3. **RootWork Canvas** - Visual, non-linear learning (Konva-based)

### Neuroscience Principles
- Trauma affects attention, working memory, executive function
- Linear instruction doesn't match all brain processing styles
- Working memory is limited in trauma-affected youth → externalize visually
- Some minds think spatially, associatively, visually
- Learning and healing are neurologically inseparable

### Target Users
- K-12 students with disabilities
- Students affected by ACEs (Adverse Childhood Experiences)
- Students with executive function challenges
- Teachers, parents, administrators
